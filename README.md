# Freedom Stack v3

A stack built upon web standards.

## Uses

- [Bun](https://bun.sh) - Mostly used for the package manager, but we can use it for the runtime
- [Hono](https://hono.dev) - Fast, lightweight, built on Web Standards. Support for any JavaScript runtime.
- [Datastar](https://data-star.dev/) - Hypermedia backend and front-end framework (takes the best of HTMX and Alpine.js)
- [Basecoat CSS](https://basecoatui.com/) - Shadcn UI-like component CSS library without the need for React
- [Bknd](https://bknd.io) - Comprehensive backend SDK with auth, db, and admin GUI

## How To

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```
